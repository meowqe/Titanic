#  Logistic Regression — Titanic Survival Prediction

## О чём этот проект

Этот проект посвящён решению классической задачи машинного обучения — предсказанию выживания пассажиров Титаника на основе их характеристик.
В качестве данных используется датасет Titanic из библиотеки seaborn, содержащий информацию о пассажирах: пол, возраст, класс билета, стоимость, наличие родственников на борту и другие социально-демографические признаки.

Главная цель проекта — показать, как с помощью анализа данных и логистической регрессии можно выявить факторы, влияющие на вероятность выживания, и построить интерпретируемую модель бинарной классификации.

---
## 1. Основная идея проекта

Проект отвечает на ключевой вопрос:

Какие характеристики пассажира влияли на его шансы выжить?

Задача формулируется как ML-проблема:

Тип задачи: Binary Classification
Целевая переменная: survived (выжил / не выжил)
Модель: Logistic Regression

Логистическая регрессия выбрана не случайно — она позволяет не только предсказывать результат, но и интерпретировать влияние признаков на вероятность выживания.

---
## 2. Что исследуется в проекте

В ходе работы анализируются:

влияние пола пассажира на выживаемость
роль класса каюты (pclass)
зависимость выживания от возраста
влияние стоимости билета (fare)
наличие семьи на борту
различия между социальными группами пассажиров

Проект показывает, что выживание зависело не случайно, а было связано с социальными и демографическими факторами.

---
## 3. ML-суть проекта

Проект демонстрирует полный pipeline Data Science:

Исследовательский анализ данных (EDA)
Очистка и подготовка признаков
Анализ взаимосвязей
Обучение модели Logistic Regression
Оценка качества классификации
Интерпретация коэффициентов модели

Особый акцент сделан на объяснимости модели — понимании почему модель принимает решения.
